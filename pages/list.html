Ordered List





A list is a collection of related items e.g. day of the week or ingredients in recipe. This page will illustrate the various types of lists we can create in a web page.




Ordered Lists


This is a collection of item in a particular order or steps that need to be followed or carried out in a given order. Some examples are given below.


Days in a Week:

  1. Sunday
  2. Monday
  3. Tuesday
  4. Wednesday
  5. Thursday
  6. Friday
  7. Saturday



Unordered Lists


This is a collection of items in no specifice order or preference, though the items are related. An example is a list of ingredients to make KFC Chicken.

Types of Nouns



Common Noun:

It is a name of an object in general, such as pencil, pen, paper, and so on.

Proper Noun:

It is the unique name of a person or a place.
